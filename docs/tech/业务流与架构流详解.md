# TestPilot · 业务流与架构流详解

> 版本：v1.0 | 日期：2026-06-17

---

## 一、端到端业务流

```
需求录入 ──→ AI解析需求 → 生成测试点 ──→ API用例设计 ──→ 用例组装 ──→ 测试执行 ──→ 结果分析 ──→ 缺陷闭环
                              ↘                ↘                ↘
                           UI用例设计      AI自动创建       性能压测
```

### 阶段详解

**阶段1：需求录入**
- 手动输入 / PDF上传解析 / Jira/TAPD同步
- 需求进入审核状态，PM确认后流转

**阶段2：AI解析需求**
- AI自动提取测试点 → 列出需覆盖的场景
- 用户可增删改，确认后进入用例设计

**阶段3：用例设计**
- API：定义接口 → 配置请求参数 → 添加断言
- Web UI：创建PO对象 → 管理元素 → 编排步骤
- APP UI：同Web UI流程，平台改为iOS/Android
- AI辅助：输入自然语言自动生成用例

**阶段4：用例组装**
- 将多个用例编入测试计划
- 选择执行环境（dev/staging）
- 配置执行策略（串行/并行/分布式）

**阶段5：测试执行**
- 触发执行 → 执行引擎调度（Celery任务队列）
- 实时结果流推送（WebSocket）
- 失败自动截图/录屏

**阶段6：结果分析**
- 执行完成后生成Allure报告
- AI分析失败用例根因
- UI用例失败时AI推荐新定位策略（自愈）

**阶段7：缺陷闭环**
- 一键创建缺陷 → 同步到Jira/TAPD
- 修复后关联缺陷→用例重新执行
- 质量门禁检查通过率/覆盖率

## 二、技术架构流（请求链路）

```
用户浏览器 (React 18 + AntD 5)
    │
    │ HTTPS (JWT Bearer Token)
    ▼
Nginx (反向代理 + 静态资源)
    │
    ├── /api/* ──→ FastAPI 后端
    │               │
    │    ┌──────────┼──────────┬──────────┬──────────┐
    │    ▼          ▼          ▼          ▼          ▼
    │  Auth      Project    TestCase   Execution   Report
    │  Service   Service    Service    Service     Service
    │    │          │          │          │          │
    │    └──────────┴──────────┴──────────┴──────────┘
    │               │
    │         ┌─────┴─────┐
    │         ▼           ▼
    │      MySQL 8.0   Redis
    │    (业务数据)   (缓存/队列)
    │         │           │
    │         ▼           ▼
    │   PostgreSQL 14  Celery Workers
    │    (分析数据)   (异步任务)
    │                    │
    │         ┌──────────┼──────────┐
    │         ▼          ▼          ▼
    │    Locust     Playwright   Appium
    │    (性能)     (Web UI)    (APP UI)
    │
    └── /ws/* ──→ WebSocket (实时推送)
                   │
                   ▼
              执行状态 / 压测指标 / AI流式输出
```

## 四、微服务拆分

| 服务 | 端口 | 职责 | 技术 |
|------|------|------|------|
| API Gateway | 8000 | 请求路由、限流、认证 | FastAPI + Nginx |
| Auth Service | 8001 | 用户认证、JWT、RBAC | FastAPI + JWT |
| Project Service | 8002 | 项目、团队、成员管理 | FastAPI |
| Case Service | 8003 | 用例、接口、PO管理 | FastAPI |
| Execution Service | 8004 | 测试执行调度 | FastAPI + Celery |
| Report Service | 8005 | 报告生成、分析 | FastAPI + Allure |
| AI Service | 8006 | AI分析、自然语言处理 | FastAPI + LangChain |
| Mock Server | 8090 | API Mock | FastAPI |
| Perf Service | 8007 | 性能压测管理 | FastAPI + Locust |
