## 三、新增原型详解（本轮 v4.0）

### P11 · 质量趋势页

**功能：** 跨版本质量数据可视化
- 4个指标卡片：平均通过率/缺陷密度/自动化覆盖率/执行总次数
- 2行4图表：通过率趋势(双Y轴) + 缺陷密度(面积图) + 失败归因(堆叠柱) + 用例增长(堆叠面积)
- 筛选器：时间范围/项目/测试类型
- 图表库：Chart.js 4.4

### P12 · 质量门禁页

**功能：** 发版准入/准出判定
- 门禁状态总览：5/6通过 → 警告
- 6项规则检查：通过率/Blocker/覆盖率/缺陷/性能/执行率
- 每项规则：绿/黄/红 指示灯 + 阈值 + 当前值 + 操作建议
- 底部操作：查看详细报告 / 创建缺陷 / 导出报告

### P13 · CI/CD集成配置页

**功能：** CI/CD Token管理与模板
- API Token管理：列表+状态+吊销
- Webhook配置：URL复制+触发事件选择
- Pipeline模板：Jenkins/GitLab CI/GitHub Actions 三选一 + 代码块
- 执行历史：来源/分支/时间/状态/门禁

### P14 · 监控部署架构图（SVG）

**功能：** 展示监控全链路拓扑
- 被监控层：Node Exporter + MySQL Exporter + cAdvisor + Redis/PG/Blackbox
- 采集层：Prometheus Server（Scrape + Rules + AlertManager）
- 告警层：飞书/企微/邮件 三通道
- 展示层：Grafana → iframe 嵌入 TestPilot
