## 二、侧边栏交互设计

### 2.1 展开/折叠逻辑

```
默认状态：所有一级菜单折叠，显示图标+文字

点击一级菜单：
  ├── 有二级子菜单 → 展开二级列表（手风琴模式，其他一级自动折叠）
  └── 无二级子菜单 → 直接跳转页面，不高亮展开状态

点击二级菜单：
  ├── 有三级子菜单 → 展开三级列表（缩进显示，其他二级自动折叠）
  └── 无三级子菜单 → 跳转页面，当前二级高亮

三级菜单 → 跳转页面，当前三级高亮
```

### 2.2 侧边栏结构

```
┌─────────────────────┐
│  🏠 TestPilot       │  ← Logo + 产品名
│  v2.0.1             │  ← 版本号
├─────────────────────┤
│  ┌─────────────────┐│
│  │ 🏠 搜索项目...   ││  ← 项目搜索/切换下拉框
│  │ ▼ 交易链路测试  ││
│  └─────────────────┘│
├─────────────────────┤
│                     │
│  📊 工作台     ✓    │  ← 一级（无子，直接跳转）
│                     │
│  📋 项目管理   ▶    │  ← 一级（有子，折叠中）
│    ├ 项目列表       │
│    ├ 需求管理       │
│    │ ├ 需求列表     │  ← 三级，缩进
│    │ └ 需求追溯     │
│    └ 测试计划       │
│                     │
│  🧪 测试用例   ▼    │  ← 一级（当前展开）
│    ├ API测试        │
│    │ ├ 接口定义 ✓   │  ← 当前页面高亮
│    │ ├ 用例列表     │
│    │ └ 用例目录     │
│    ├ Web UI测试  ▶  │
│    ├ APP UI测试  ▶  │
│    └ 用例组装工作台 │
│                     │
│  ⚡ 测试执行   ▶    │
│  🔥 性能压测   ▶    │
│  📊 测试报告   ▶    │
│  🤖 AI助手     ▶    │
│                     │
│  ───────────────    │  ← 分割线
│  ⚙️ 系统管理   ▶    │  ← 仅admin/团队owner可见
│                     │
├─────────────────────┤
│  👤 狼图腾OPC    │  ← 用户头像+名字
│  超级管理员        │  ← 角色
└─────────────────────┘
```

### 2.3 面包屑导航

```
每个页面顶部显示完整路径：

接口定义页面：  🧪 测试用例 > API测试 > 接口定义
链路监控页面：  🔥 性能压测 > 监控大盘 > 链路监控
用户列表页面：  ⚙️ 系统管理 > 用户管理 > 用户列表
```

## 四、顶部导航栏

```
┌───────────────────────────────────────────────────────────┐
│  ≡ (折叠侧栏)  面包屑路径...              🔔 ⚙️ 👤 狼图腾 │
└───────────────────────────────────────────────────────────┘

🔔 = 通知中心（执行完成/失败告警/系统公告）
⚙️ = 个人设置（修改密码/语言切换/API Token管理）
👤 = 用户菜单（个人信息/切换团队/退出登录）
```
