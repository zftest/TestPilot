## MVP 范围总结

```
Phase 0 (Week 1-2)： ← 基础设施
  ✅ 脚手架 + 认证 + 布局 + 用户管理

Phase 1 (Week 3-6)： ← MVP核心闭环
  ✅ 团队/项目 → API接口定义 → 测试用例 → 测试计划 → 执行引擎 → 结果/报告

Phase 2 (Week 7-10)： ← AI增强
  (先不做) LangChain集成 + 自然语言创建 + 失败分析 + 自愈

Phase 3 (Week 11-14)： ← UI/性能
  (先不做) Playwright + Appium + Locust + 监控大盘

Phase 4 (Week 15-18)： ← 企业功能
  (先不做) CI/CD + 缺陷管理 + 权限系统 + 审计日志
```
